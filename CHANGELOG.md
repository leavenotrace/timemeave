格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

- 增强的错误处理和加载状态管理
- 移动端优化的响应式设计
- 触摸交互支持（滑动、长按、下拉刷新）

- 导航栏支持移动端折叠菜单

- 增强了错误提示的友好性
- 改进了触摸设备的交互体验

### 技术改进
- 使用 TypeScript 确保类型安全
- 实现了 Provider 模式的状态管理
- 添加了自定义 Hooks 用于复用逻辑

- 支持 Next.js 13+ App Router
- 添加了完整的错误监控系统

## 版本说明

### 语义化版本格式
- **主版本号**：当你做了不兼容的 API 修改
- **次版本号**：当你做了向下兼容的功能性新增
- **修订号**：当你做了向下兼容的问题修正

### 变更类型
- **新增** - 新功能
- **改进** - 对现有功能的改进
- **修复** - Bug 修复
- **移除** - 移除的功能
- **安全** - 安全相关的修复
- **废弃** - 即将移除的功能

### 发布周期
- **主版本**：重大架构变更或破坏性更新
- **次版本**：新功能发布，通常每月一次
- **修订版本**：Bug 修复，根据需要发布
